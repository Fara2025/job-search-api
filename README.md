# Job Search API

API-приложение для поиска работы, разработанное с использованием Python и FastAPI.

## Описание проекта
Данный проект представляет собой серверное API-приложение, предназначенное для работы с вакансиями, работодателями и резюме соискателей.
API поддерживает основные CRUD-операции и фильтрацию данных.

## Используемые технологии
- Python
- FastAPI
- Pydantic
- Uvicorn

## Основные возможности
- Получение списка вакансий
- Создание, обновление и удаление вакансий
- Работа с работодателями
- Создание и просмотр резюме
- Фильтрация вакансий по зарплате и ключевым словам
- Автоматическая документация Swagger (OpenAPI)

## Запуск проекта локально
```bash
pip install -r requirements.txt
uvicorn main:app --reload
